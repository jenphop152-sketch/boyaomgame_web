--------------------------------------------------------------------------------
                            TYPES OF INQUIRIES
--------------------------------------------------------------------------------

🐛 BUG REPORTS
   Found a bug? Please contact us with:
   - Description of the problem
   - Steps to reproduce
   - Expected vs actual behavior
   - Screenshots (if applicable)

💡 FEATURE REQUESTS
   Have an idea to improve UserLooker? We welcome suggestions!
   - Describe your idea and its use case
   - Explain how it would benefit users

🔐 PRIVACY REQUESTS
   For privacy-related requests (data access, correction, deletion):
   - Contact the developer directly via Discord
   - Include your Discord User ID for verification
   - Specify the type of request (access/correction/deletion)

⚠️ SECURITY ISSUES
   Found a security vulnerability?
   - Please contact the developer directly and privately
   - Do not share vulnerability details publicly
   - Allow time for the issue to be resolved before any disclosure

❓ GENERAL QUESTIONS
   For general questions about the project:
   - Check if your question has been answered previously
   - Contact the developer via Discord

--------------------------------------------------------------------------------
                            RESPONSE TIME
--------------------------------------------------------------------------------

We aim to respond to inquiries within:

   - Bug Reports:        1-3 business days
   - Feature Requests:   3-7 business days  
   - Privacy Requests:   1-5 business days
   - Security Issues:    24-48 hours

Note: Response times may vary based on complexity and developer availability.
