================================================================================
                     FEATURE BRIEFING: LOADING SKELETON
================================================================================
Category: Frontend
Priority: Low
Complexity: Low

--------------------------------------------------------------------------------
OVERVIEW
--------------------------------------------------------------------------------

Replace plain "Loading..." text with skeleton loading components that mimic
the layout of the actual content. This provides better user experience by
showing the structure of upcoming content.

--------------------------------------------------------------------------------
REQUIREMENTS
--------------------------------------------------------------------------------

1. Create skeleton components for:
   - User profile card (Result page)
   - Discord accounts list
   - Message cards
   - Rank history timeline
   - Statistics cards

2. Skeleton styling:
   - Animated shimmer effect
   - Gray placeholder boxes
   - Same dimensions as real content

3. Apply to all pages with async data loading

--------------------------------------------------------------------------------
DEPENDENCIES
--------------------------------------------------------------------------------

- None (pure CSS/React feature)

--------------------------------------------------------------------------------
UI/UX CONSIDERATIONS
--------------------------------------------------------------------------------

- Skeletons should match actual content layout closely
- Animation should be subtle, not distracting
- Duration should feel appropriate (not too fast/slow)

--------------------------------------------------------------------------------
FILES TO CREATE/MODIFY
--------------------------------------------------------------------------------

NEW FILES:
  - frontend/src/components/Skeleton.jsx (base component)
  - frontend/src/components/skeletons/ProfileSkeleton.jsx
  - frontend/src/components/skeletons/MessageSkeleton.jsx
  - frontend/src/styles/skeleton.css

MODIFIED FILES:
  - frontend/src/components/Result.jsx (use ProfileSkeleton)
  - frontend/src/components/MessageViewer.jsx (use MessageSkeleton)

--------------------------------------------------------------------------------
IMPLEMENTATION APPROACH
--------------------------------------------------------------------------------

CSS Animation:
```css
.skeleton {
  background: linear-gradient(90deg, #1e293b 25%, #334155 50%, #1e293b 75%);
  background-size: 200% 100%;
  animation: shimmer 1.5s infinite;
}

@keyframes shimmer {
  0% { background-position: 200% 0; }
  100% { background-position: -200% 0; }
}
```

--------------------------------------------------------------------------------
ACCEPTANCE CRITERIA
--------------------------------------------------------------------------------

[ ] All loading states use skeleton components
[ ] Skeletons match content layout
[ ] Shimmer animation is smooth
[ ] Skeletons work in both dark/light modes
[ ] No layout shift when content loads
